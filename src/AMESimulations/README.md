# Solve QUBOs by simulating quantum annealing

Code to simulate the quantum annealing process using explicit time-dependent simulations of the quantum dynamics. 
This is quite slow, but should eventually allow access to some experimental quantities not visible otherwise.

## Using the code

To run the most basic version of the solver, use the command:
```
julia SolveQUBO.jl problem_instance.pkl 
```
Where the problem instance is a QUBO generated by the ```make_qubos.py``` code.

This will simulate the quantum annealing process with a Schrodinger equation solver and a very short annealing time of 10ns, good for testing that the code is working with small problems which will take a minute or two. 
By default, it will print the most likely outputs to the terminal and generate a PDF file containing a plot of the populations over time.

Command line options exist to:
 - ```--name```: Set a name for the problem (used for output files)
 - ```--solve-ame```: Switch to simulations using the adiabatic master equation
 - ```--plot-energies```: Output a plot of the eigenenergies of the Ising model as a function of the annealing paramter
 - ```--annealing-time```: Set the annealing time
 - ```--sparse```: Use sparse matrix representation in computation of ... 

Use ```julia SolveQUBO.jl --help``` to see details.
